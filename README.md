Flask-Airbrake
===============================

version number: 0.0.1
author: Piotr Poteralski

Overview
--------

Flask extension for Airbrake

Installation / Usage
--------------------

To install use pip:

    $ pip install Flask-Airbrake


Or clone the repo:

    $ git clone https://github.com/rstit/flask-airbrake.git
    $ python setup.py install
